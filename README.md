
# daysnap-sass-define

> sass 样式变量、工具库

## 安装

```shell
npm install @daysnap/define-sass
```

## 使用

```scss
@import "~@daysnap/define-sass/define";
```

## 开发

1. 克隆 `git@github.com:daysnap/daysnap-define-sass.git`

```shell
git clone git@github.com:daysnap/daysnap-define-sass.git
```

2. 安装依赖

```shell
npm install
```
